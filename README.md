# Signals and Systems

Repositorio personal de experimentos y pequeñas implementaciones
relacionadas con **Signals and Systems** (Oppenheim).

Basado en el curso del MIT OpenCourseWare.
Objetivo: práctica diaria, exploración y registro personal.

## Contenido
- Señales seno/coseno
- Exponenciales complejas
- Unit step, Unit impulse
- Ventanas y operaciones básicas
- Ploteos en tiempo continuo y discreto
- Convolución
